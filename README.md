# scholar-search

面向计算机学科研究生的全能学术搜索与评估 Skill。

## 六种模式

| 模式 | 功能 | 示例 |
|------|------|------|
| **C** 团队搜索 | 团队结构 + 产出模式 + 定位分析 | "分析一下 XX 课题组" |
| **D** 方向发现 | 必读论文 + 核心研究者 + GitHub代码 + 入门路径 | "我研一想做深度哈希，该读什么？" |
| **E** 复现评估 | 代码质量 + 硬件需求 + 性价比评分 | "这篇论文代码能跑吗？" |
| **F** 基准追踪 | 标准数据集 + SOTA排行榜 + 天花板判断 | "这个方向SOTA多少？天花板到了吗？" |
| **G** 招生匹配 | 招生导师 + 课题组风格 + 择导建议 | "做XX方向的哪些老师在招学生？" |
| **H** Socratic引导 | 5层对话：模糊想法→具体行动计划 | "我对联邦学习有兴趣但不知道具体研究什么" |

## 知识增强

- 三遍阅读法（Survey → Understanding → Deep Dive）
- 引文链追踪（向前/向后引用分析）
- 来源质量矩阵（期刊/引用/复现/独立性/时效五维评估）

## 安装

将 `SKILL.md` 放入 Claude Code 的 skills 目录即可。

## 来源

基于与 Claude Code 多轮对话迭代开发，从真实的 CS 研究生需求出发设计。
